Hugo로 Github 블로그 만들기

Posted Jan 29, 2019

개인 블로그 필요성을 느껴 블로그를 만들기 위해 상당히 많은 시간을 투자했다. 대게는 naver 블로그, tistory 등을 많이 사용하지만 만드는 김에 Git Hosting을 사용해보고자 했다. Git Hosting을 이용하여 블로그를 만들기 위해서는 Static Site Generator를 사용해야한다.

Static Site Generator란?

추후에 설명을 추가하도록 하겠다. 보통 많이 사용하는 Static Site Generator에는 Jekyll , Hexo, Hugo가 있다. 각 생성기를 간단하게 설명하자면

  • Jekyll
    • 루비 기반
    • 가장 많이 사용되고 있어 참조할 한글 문서들 많음
    • 포스팅 수가 많아지면 속도가 느려진다는 말이 있음
    • 윈도우 공식 지원 안된다

  • Hexo

    • 자바스크립트(Node.js) 기반
    • 개발자가 중국인

  • Hugo

    • Golang 기반
    • 빠른 빌드
    • 문서화 잘 되어있음(영어지만..)
